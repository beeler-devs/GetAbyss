# Get Abyss — VoiceIDE

**Your voice-first assistant — for coding, email, calendar, and more.**

Website for [get-abyss.app](https://get-abyss.app).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Optimized for [Vercel](https://vercel.com). Connect this repo for automatic deployments.

## Tech

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Default dark mode
